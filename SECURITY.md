## Reporting a Vulnerability

We take the security of ASAP Protocol seriously. If you have found a security vulnerability, please do NOT report it publicly.

### How to Report

Please email us at [security@asap-protocol.org](mailto:security@asap-protocol.org) with the details of the vulnerability. We will try to get back to you within 48 hours.

### Information to Include

Please include as much information as possible to help us reproduce the issue:

- Steps to reproduce the vulnerability
- Version of the library used
- Python version
- Operating system
- Any relevant logs or output

### What Happens Next

1. We will acknowledge your report within 48 hours.
2. We will investigate the issue and confirm if it is a vulnerability.
3. We will work on a fix and release it as soon as possible.
4. We will credit you for the discovery (unless you prefer to remain anonymous) once the fix is released.

Thank you for helping keep the ASAP Protocol secure!
